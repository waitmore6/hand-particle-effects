# 🌌 Hand-Particle-Interaction (手势交互粒子特效系统)

基于 AI 前端视觉识别与 HTML5 Canvas 高性能物理引擎的实时流光交互项目。无需任何后端服务器，完全在浏览器本地端运行。

## ✨ 核心亮点
- **智能手势识别**：基于 MediaPipe Hands 毫秒级追踪手部 21 个骨骼节点。
- **炫彩动态反馈**：
  - ✊ **握拳**：触发重金属耀眼流金色冲击波，并同步释放纯 JS 合成的科幻引爆音效。
  - ✋ **张开**：释放具有呼吸感的荧光橙色脉冲星云。
  - ✌️ **V手势**：粒子化为未来通透电光蓝的量子风暴。
  - ☝️ **单指吸附**：全屏粒子向指尖凝聚，并触发 1 秒超长星空流光拖尾。
- **极致性能优化**：通过低复杂度模型锁死、平方代替开方算法，将 1200 个粒子的物理碰撞计算完美适配低配办公轻薄本。
